# HTTP/2 streams

## Evidence
Batch lesson 6 maps earlier labels/chunks/frame intuition onto HTTP/2 stream IDs and frames.

## Challenges
Potential confusion: HTTP/2 multiplexing improves request sharing but does not eliminate all head-of-line blocking, especially beneath it at TCP packet loss level.

## Engagement Pattern
Learner likely benefits from seeing old mechanism with new protocol names.

## Adjust
If reviewing, ask learner to explain stream ID as channel label before discussing browser performance details.
